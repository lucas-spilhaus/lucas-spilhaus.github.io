Understanding Margin, Border, and Padding

Overview

Any HTML element has what is effectively a box wrapped around it. This box is made up of four parts; padding, border, margin, and the content itself. We call this the Box Model.

Padding

Padding is the area inside the element and lays between the content and border. By increasing padding, the content's background gets larger.

Border

Border is the area surrounding the element and lays between the padding and margin. Border is incredibly simple, just being a, well, border around the element. Despite how simple it is, border is rather flexible, having various different styles to be chosen from to match whatever is needed.

Margin

Margin is the final part of the Box Model and surrounds the border. Margin is also quite simple to understand, it's basically a gap between other elements. For example, if you set your margin to 50px, there would be a 50px gap around the main element and any other elements placed beside it.
